# Практические задания по многопоточности

Этот проект содержит 8 оконных приложений для работы с многопоточностью и асинхронностью в C#.

## Требования

- .NET 6.0 или выше
- Windows (для Windows Forms)

## Сборка проекта

```bash
dotnet build
```

## Запуск проекта

```bash
dotnet run
```

## Описание заданий

### Задание 1: Генерация чисел, букв и символов
Приложение запускает три потока:
- Поток генерации чисел (0-99)
- Поток генерации букв (A-Z)
- Поток генерации символов (!@#$%...)

Каждый поток имеет настраиваемый приоритет (Lowest, BelowNormal, Normal, AboveNormal, Highest). Результаты выводятся в текстовое поле.

### Задание 2: Копирование больших файлов
Приложение позволяет:
- Выбрать исходный файл
- Выбрать путь для копирования
- Указать количество потоков для копирования
- Отслеживать прогресс копирования через прогресс-бар

### Задание 3: Копирование файлов с паузой и остановкой
Расширение задания 2 с возможностями:
- Приостановка копирования (кнопка "Пауза")
- Полная остановка копирования (кнопка "Стоп")
- После остановки можно выбрать другой файл для копирования

### Задание 4: Копирование директорий
Приложение для копирования целых директорий:
- Выбор исходной директории
- Выбор пути для копирования
- Указание количества потоков
- Отображение прогресса копирования всех файлов

### Задание 5: Копирование директорий с паузой и остановкой
Расширение задания 4 с возможностями:
- Приостановка копирования
- Полная остановка копирования
- После остановки можно выбрать другую папку для копирования

### Задание 6: Подсчет факториала
Асинхронное вычисление факториала числа:
- Ввод числа
- Асинхронный подсчет факториала
- Отображение результата (поддерживает большие числа через BigInteger)

### Задание 7: Подсчет степени числа
Асинхронное вычисление степени числа:
- Ввод числа и степени
- Асинхронный подсчет степени
- Отображение результата

### Задание 8: Подсчет символов в тексте
Асинхронный анализ текста:
- Ввод текста
- Подсчет гласных букв
- Подсчет согласных букв
- Подсчет общего количества символов

- `Task3Form.cs` - Задание 3
- `Task4Form.cs` - Задание 4
- `Task5Form.cs` - Задание 5
- `Task6Form.cs` - Задание 6
- `Task7Form.cs` - Задание 7
- `Task8Form.cs` - Задание 8

## Используемые технологии

- C# .NET 6.0
- Windows Forms
- Threading (Thread, ManualResetEvent)
